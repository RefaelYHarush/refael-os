# RAFAEL OS – תוכנית שיפור כמה רמות קדימה

מסמך זה מתאר שיפור של האפליקציה בחמש רמות: מארכיטקטורה וקוד, דרך נתונים ואחסון, פיצ'רים אמיתיים, חוויית משתמש, ועד מוצר מלא.

---

## רמה 1 – ארכיטקטורה וקוד (קריטי) ✅ הושלם

**מטרה:** קוד נקי, ניתן לתחזוקה והרחבה.

| פעולה | תיאור |
|--------|--------|
| פיצול לרכיבים | `Layout`, `DashboardView`, `TradingView`, `SaasView`, `VisionView`, ורכיבים משותפים: `Card`, `Badge`, `ProgressBar`, `PnlChart` |
| מודול נתונים | `src/data/constants.js` – ערכי ברירת מחדל ו־storage keys |
| State מרוכז | `AppContext` עם usePersistedState – XP, משימות, עסקאות, פרויקטים, חזון |
| טיפוסים (אופציונלי) | מעבר ל־TypeScript או JSDoc בהמשך |

---

## רמה 2 – נתונים ואחסון ✅ חלקי

**מטרה:** הנתונים נשמרים ולא נעלמים ברענון.

| פעולה | תיאור |
|--------|--------|
| Persistence | ✅ localStorage לכל ה־state הרלוונטי |
| "עסקה חדשה" | ✅ מודל עם שדות + שמירה לרשימה ו־storage |
| "משימה חדשה" | ✅ כפתור + משימה יומית עם XP |
| Export/Backup | כפתור ייצוא JSON – להמשך |

---

## רמה 3 – פיצ'רים אמיתיים

**מטרה:** כל אלמנט בממשק עושה משהו אמיתי.

| פעולה | תיאור |
|--------|--------|
| Deep Work Timer | טיימר פומודורו עם התחלה/עצירה ושמירת היסטוריה |
| Rethink יומי | חישוב דינמי לפי win rate ו־trades אתמול |
| מערכת XP ורמות | חוקים ברורים: כמה XP לרמה, בונוסים לסטריק |
| לוח שנה במסחר | לחיצה על יום = רשימת עסקאות או הוספת עסקה |
| SaaS Kanban | גרירה בין עמודות + "פרויקט חדש" פונקציונלי |

---

## רמה 4 – חוויית משתמש

**מטרה:** אפליקציה נוחה, מהירה ונגישה.

| פעולה | תיאור |
|--------|--------|
| PWA | manifest + Service Worker – התקנה על המסך |
| Theme | Toggle כהה/בהיר עם שמירה |
| אנימציות | View transitions בין טאבים |
| מובייל | המבורגר, כרטיסים נוחים למגע |
| נגישות | מקלדת, aria-labels, ניגודיות |

---

## רמה 5 – מוצר וסנכרון

**מטרה:** RAFAEL OS כמוצר עם נתונים בין מכשירים.

| פעולה | תיאור |
|--------|--------|
| Backend | Supabase/Firebase – users, trades, tasks, projects |
| Auth | התחברות – נתונים לפי משתמש |
| סנכרון | אותו משתמש במכשיר אחר רואה אותם נתונים |
| אנליטיקה | גרפים לפי חודש/שנה, השוואות setups |
| התראות | תזכורות (דורש PWA/שרת) |

---
