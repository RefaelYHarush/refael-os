צלם מסך 1–2 דקות לפי השלבים למטה, העלה ל־YouTube (Unlisted), והדבק את הקישור בטופס.

## צ'קליסט לפני הצילום

- [ ] הדפדפן פתוח בחלון נקי (או טאב אחד).
- [ ] רזולוציה סבירה (למשל 1280×720) – לא חובה אבל נראה טוב.
- [ ] מיקרופון כבוי (אין חובה לדיבוב – מספיק להראות את המסך).
- [ ] האתר: https://refael-os.vercel.app (או localhost אם מציגים פיתוח).

## אחרי הצילום

1. העלה את הקובץ ל־YouTube.
2. הגדר ל־**Unlisted** (לא פרטי – גוגל צריכה לגשת לקישור).
3. העתק את קישור הסרטון (למשל `https://www.youtube.com/watch?v=...`).
4. בטופס אימות גוגל: בשדה **Demo video** / **Video showing app functionality** – הדבק את הקישור.
5. שלח את הטופס (Submit for verification).

## טיפים

- אם טעית – אפשר לחתוך ב־YouTube או לצלם שוב; 1–2 דקות מספיקות.
- אין צורך באפקטים או מוזיקה – מספיק להראות את הזרימה: נחיתה → התחברות גוגל → דשבורד → יומן.
- אם יש בעיית הרשאות (למשל "insufficient scopes") – עדיף לתקן לפני הצילום או להזכיר בסרטון שמופיעה הודעה ויש "התחבר שוב עם הרשאות מלאות".
