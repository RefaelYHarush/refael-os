# REFAEL OS

Life Management System – דשבורד אישי לניהול מסחר, SaaS, משימות יומיות וחזון ארוך טווח.

## הרצה

```bash
cd refael-os
npm install
npm run dev
```

פותח בדפדפן: http://localhost:5173

## טכנולוגיות

- React 18 + Vite
- Tailwind CSS (כולל dark mode)
- Lucide React (אייקונים)
- RTL (עברית)
